HR Analytics Report

A comprehensive Power BI report for organisational analytics

← Back to Portfolio

Project Overview

This is a fully fledged report containing an in-depth analysis of an organisation's workforce efficiency, employee retention, cost management, and productivity.

Key Features

  • Assess Employee Performance
  • Analyse Workforce Engagement & Satisfaction
  • Track Training Effectiveness
  • Monitor Employee Turnover

Technical Implementation

The dashboard was built using Power BI, incorporating data from the organisation's on-site database, quering all the data using SQL. Data cleaning was done through the use of Python and its Pandas library. Advanced DAX formulas were used for complex calculations and metrics, while custom visualizations were created to present the data in an intuitive and actionable format with a theme that matches the organisation's colours.
